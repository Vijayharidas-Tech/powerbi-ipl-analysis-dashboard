# 🏏 IPL Analysis Dashboard (2008–2025) | Power BI

## 📌 Project Overview

This project is an interactive Power BI dashboard that analyzes Indian Premier League (IPL) data from **2008 to 2025**. The dashboard provides insights into team performance, player statistics, match outcomes, venues, and season trends using interactive visualizations.

---

## 🎯 Objectives

- Analyze IPL seasons from 2008–2025.
- Compare team performances across seasons.
- Identify top-performing players.
- Explore venue and match-winning trends.
- Build an interactive dashboard for cricket analytics.

---

## 📊 Dashboard Features

- 🏆 Season-wise Champions
- 📈 Team Performance Analysis
- 🏏 Top Run Scorers
- 🎯 Top Wicket Takers
- 🏟️ Venue Analysis
- 📅 Season-wise Match Statistics
- 📊 Toss Decision Analysis
- 🎛️ Interactive Filters and Slicers
